# Restaurant Sales Dashboard

##Project Overview
This project focuses on analyzing restaurant sales and order performance using Python, SQL, and Power BI.  
It provides insights into revenue trends, customer behavior, top-selling items, and order types to support
data-driven business decisions.

---

##Tools & Technologies
- *Python (Pandas)* – Data cleaning & preprocessing  
- *SQL* – Analytical queries and business insights  
- *Power BI* – Interactive dashboard & visualizations  
- *GitHub* – Project version control and documentation  

---

## Project Workflow
1. Raw restaurant order, customer, and menu data collected  
2. Data cleaned using Python (handling missing values, duplicates, and datatypes)  
3. SQL queries written to analyze sales, orders, and item performance  
4. Interactive Power BI dashboard built for visualization and insights  

---

## Key Insights
- April recorded the highest number of orders, indicating strong monthly growth  
- Main course items contribute the highest share of total revenue  
- Online and Dine-in orders contribute almost equally to total sales  
- OMR location shows the highest sales compared to other branches
